# Log Redaction Frontend

Simple React frontend for the log redaction service.

## Setup

```bash
cd frontend
npm install
```

## Development

Start the backend server first (from project root):
```bash
npm start
```

Then start the frontend dev server:
```bash
cd frontend
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
```

The build output will be in `dist/` folder.
